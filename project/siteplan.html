WDD 131 - Project Site Plan

BHC - Body Health Care

This name represents the purpose of the entire site. Which is to give users important information about body health care.

Site Purpose

The site will contain a BMI (Body Mass Index) calculator. In which the user will be able to calculate their own BMI. Also they will have information about the different results that they can get.
Also there they will find another health care information like good habits for the body and a form to the user send comments about the site or questions.

Scenarios

  • How can I improve my BMI percentage?
  • Where can I ask an specific question?
